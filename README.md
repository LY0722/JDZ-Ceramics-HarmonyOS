# 景德镇陶瓷文化展示系统 (JDZ Ceramics HarmonyOS App)

## 系统简介
本项目是一款基于 HarmonyOS (鸿蒙) 开发的陶瓷文化展示应用。通过精美的界面和流畅的交互，向用户展示景德镇悠久的陶瓷历史、精湛的制作工艺以及代表性的陶瓷作品。

## 技术栈
- **开发框架**: HarmonyOS SDK
- **开发语言**: ArkTS
- **构建工具**: Hvigor
- **设计规范**: HarmonyOS Design

## 项目结构
- `AppScope/`: 应用全局资源与配置
- `entry/`: 应用主模块，包含所有页面与逻辑
- `hvigor/`: 构建配置相关

## 系统要求
- DevEco Studio 3.1+
- HarmonyOS SDK API 9+

## 运行指南
1. 下载并安装 [DevEco Studio](https://developer.harmonyos.com/cn/develop/deveco-studio)。
2. 使用 DevEco Studio 打开 `JDZCramics` 文件夹。
3. 等待 `oh-package` 依赖自动下载完成。
4. 连接鸿蒙真机或启动模拟器，点击运行。

## 主要功能
- **文化百科**: 介绍陶瓷历史背景。
- **精品展示**: 高清图文展示陶瓷名作。
- **互动体验**: 模拟陶瓷制作过程或 VR 展示。
- **个人中心**: 管理收藏与反馈。
